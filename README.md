## Deployment

### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]

### Setting Up the Project

Install dependencies locally:

```bash
npm install
```

Run the application:

```bash
npm start
```

Test the application:

```bash
npm test
```

[node.js]: https://nodejs.org/
[npm]: https://www.npmjs.com/get-npm
